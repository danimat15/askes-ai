# AskES AI

This project is an AI-powered Next.js application designed to provide intelligent assistance, particularly in the health domain. It features:

-   **Next.js Framework**: A modern React framework for building full-stack web applications.
-   **Authentication**: Integrated user authentication system (likely Clerk).
-   **Interactive Chat**: A dynamic chat interface for user interaction.
-   **Voice Interaction**: Capabilities for voice-based communication.
-   **Dashboard**: A user dashboard for managing interactions and information.
-   **Comprehensive API**: Backend API routes for chat, session management, report generation, user handling, and specialist suggestions.
-   **Knowledge Base (RAG)**: Utilizes a Retrieval Augmented Generation (RAG) system with a knowledge base containing PDF documents related to dental health (`gigi`), skin health (`kesehatan-kulit`), and general health (`kesehatan-umum`).
-   **Database**: Uses Drizzle ORM for database interactions.
-   **Modular Components**: Organized with reusable UI components and utility functions.

This application aims to leverage AI to provide informative and interactive experiences, particularly in healthcare consultations.
